# Tempo Strike

A Rhythm game powered by React, Three.js (R3F), and MediaPipe Hand Tracking.

## Requirements

- Node.js (v16+)
- pnpm (recommended) or npm

## Installation

1. Clone this repository.
2. Install dependencies:

```bash
pnpm install
# or
npm install
```

## Running Locally

```bash
pnpm dev
# or
npm run dev
```

Open your browser at `http://localhost:5173`.

## Controls

- Allow Camera permissions when prompted.
- Stand back ~1.5 - 2 meters from the camera so your upper body and hands are visible.
- Use your hands to slash the boxes in the direction of the arrows (or any direction for dots).
- **Red** = Left Hand
- **Blue** = Right Hand

## Assets

The game streams music from a demo URL. You can upload your own MP3/OGG files in the menu.
